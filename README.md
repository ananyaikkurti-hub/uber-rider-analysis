## Project Overview
This project analyzes Uber ride data to understand usage patterns, trip purposes,
and distance trends using Python and Jupyter Lab.

## Dataset
Uber Drives dataset containing trip start/end times, locations, distance,
category (Business/Personal), and trip purpose.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Lab

## Key Analysis
- Datetime preprocessing and feature engineering
- Peak hour and weekday analysis
- Business vs personal trip comparison
- Distance and purpose-based insights
- Start and stop location frequency analysis

## Results
The analysis highlights that Uber usage is primarily business-driven,
with most trips being short-distance and recurring between specific locations.
